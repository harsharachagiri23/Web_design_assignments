## Tech

 Wandering Earl uses a number of HTML controls to work properly:

- [href] - creates a hyperlink to web pages, files, email addresses, locations.
- [a] - An anchor tag is a HTML element that creates a link to a target URL.
- [div] - The <div> tag defines a division or a section in an HTML document.
- [table] - HTML tables allow web developers to arrange data into rows and columns.
- [img] - The <img> tag is used to embed an image images are linked to web pages.
- [form] - The <form> tag is used to create an HTML form for user input.
- [button] - The <button> tag defines a clickable button.
